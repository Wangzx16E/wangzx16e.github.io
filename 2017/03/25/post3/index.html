使用Twitter API获取数据

发表于 2017-03-25

自己在用的时候，没有找到针对Github中已有的wrapper使用说明，于是自己简单写了一下。

  1. 在Github选择相应语言的API wrapper，这里我选择的是python版本bear/python-twitter

  2. 按照wrapper给的步骤安装包，配置好。

  3. 在Twitter开发者中建立自己的app (有些wrapper里会给出步骤), 获取
    consumer_key, consumer_secret ; access_token_key, access_token_secret。
    (后面这2个至少可以用自己的账户获取)

  4. 检查自己的VPN设置，比如我用的如下图：

    确保IP地址只有一个，不然会报错。

    设置完成后，再运行，即可认证。

备注：Facebook，Foursquare等在Github上都有wrapper使用API，设置好了VPN就可以使用了：）
